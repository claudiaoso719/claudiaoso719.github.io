Hot Reloading System for Game Engines

Built a real-time script hot reloading pipeline using Lua C API and rolling hash algorithm, enabling developers to see code changes instantly without application restarts. System uses a Script Registry to track files and detect changes during the game's update loop, eliminating iteration friction during development.

Computational Jigsaw Puzzle Solver

Built a computer vision system that automatically solves jigsaw puzzles by extracting piece contours, matching edges using Mean Squared Error analysis, and assembling solutions through greedy graph traversal. Implemented rotation handling with perspective transformation and achieved sub-second assembly times.
